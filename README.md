# Hello world!

Let's collaborate! Each student will add a page to this website.

## Instructions

1. Do not make changes to the main branch.
2. Create a branch named after yourself.
3. In that branch, create a folder names after yourself.
4. In that folder, create a page.
   - Your page should at least have the files `index.html` and `style.css`
5. Include (described below):
   - Greeting
   - Origin story
   - Image
   - Attribution
   - Link back to home page.
6. Use flexbox to organize your page structure.
7. Add your page to the home page.
   - Edit the home page `index.html` and add a link to your page with the tags `<li>` and `<a>`
8. Commit your changes.
   - Include proper commit messages.

## Stick to the theme

- Try to stick to the theme of Hello World!
- The content will include your introduction to the digital world.
- Include a greeting. Examples:
  - Hello, I'm John!
  - Hello from Tennessee!
- Include an origin story.
  - A short paragraph about how you first "entered" the world of technology or your first memory of using a computer.
- Include at least one image.
  - An image that represents your world (a photo of your workspace, a favorite hobby, literal map of where you are, etc).
- Include attributions.
- Include a link back to the home page.
  - Every page must have a link at the bottom that says Back to the World which points back to ../index.html.
